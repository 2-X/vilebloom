-- ubuntu for windows styling --

How to set the Vilebloom color palette for the Ubuntu for Windows shell or Command Prompt

# EASY WAY INSTRUCTIONS #
1. copy the shortcut named "Terminal". note that windows saves color and layout settings in shortcuts!
2. redirect it to your Command Prompt or Ubuntu for Windows. when you open them using this shortcut, it will be styled correctly!

# HARD WAY INSTRUCTIONS #
1. open the Ubuntu for Windows shell or Command Prompt
2. right-click the top bar
3. select "Properties"
4. go to the "Colors" tab
5. there are 16 colors in the color palette. each corresponds to coloring some part of the shell.
   the following colors are laid out in the same order as they are meant to go in the palette.
   click on each of them and change the rgb values to match the given values below

- background: rgb(22,39,51)
- highlighted text foreground: rgb(44,68,89)
